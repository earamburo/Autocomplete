# Autocomplete Client Search

React autocomplete component with MUI that fetches and displays client data.

## Prerequisites

- Node.js 18+
- npm
- git clone https://github.com/earamburo/Autocomplete.git

## Installation
```bash
npm install
```

## Run Locally
```bash
npm run dev
```

Open suggested localhost

## Run Tests
```bash
npm test
```

## Features

- Autocomplete search for clients
- Fetches data from JSONPlaceholder API
- Formats names as "Last, First, (Suffix)"
- Displays client address on selection
